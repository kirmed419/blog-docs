Getting Started: How to install Python, Jupyter and important Data Science packages.

Learn how Python, Jupyter and important Data Science packages are installed.
Kirouane Mohammed

Guide to Installing Python and Data Science Packages#

Step 1: Install Python#

Before you can start using Python for data science, you need to have Python installed on your system. Here’s how to do it:

Windows#

  1. Download the Installer: Go to the official Python website at python.org and download the latest version of Python for Windows.
  2. Run the Installer: Double-click the downloaded installer to start the installation process.
  3. Customize Installation: When prompted, make sure to check the box that says “Add Python to PATH.” This step is crucial because it allows you to use Python from the command line. You can then click on “Install Now” or choose “Customize installation” to select additional features.
  4. Verify Installation: After installation, open Command Prompt and type python --version. If Python is installed correctly, it will display the version number.

Step 2: Install pip (Python Package Manager)#

pip is the package manager for Python, allowing you to easily install and manage Python packages.

Windows, macOS, and Linux#

  • pip comes pre-installed with Python 3.4 and later. To verify if pip is installed, open your command line or terminal and type: 
    pip --version

If pip is not installed, you can install it by following the official pip installation guide.

Step 3: Install Data Science Packages#

To install pandas, numpy, scipy, and matplotlib on Windows, you can use the following commands. These commands should be executed in the command prompt or PowerShell.

  1. First, make sure you have Python installed. You can check this by typing: python –version

  2. Using pip, you can install the packages. Use the following command to install pandas, numpy, scipy, and matplotlib:

pip install pandas numpy scipy matplotlib

If there are no errors when importing, the installation was successful.

Step 4: Install Jupyter#

What is Jupyter?#

Jupyter is an open-source project that provides an interactive computing environment where you can create and share documents, called “notebooks,” that contain live code, equations, visualizations, and explanatory text. It supports multiple programming languages, with Python being the most popular.

What is Jupyter Used For?#

Jupyter is commonly used in data science, scientific computing, and machine learning for:

  1. Data Cleaning and Transformation: Easily manipulate and clean datasets.
  2. Numerical Simulation: Perform calculations and simulations.
  3. Data Visualization: Create plots and visual representations of data.
  4. Machine Learning: Build, train, and evaluate machine learning models.
  5. Interactive Demonstrations: Provide interactive demos of code and results.
  6. Educational Purposes: Teaching and learning programming and data science concepts.

How to Install Jupyter on Windows#

  1. Install Jupyter using pip:

    • Open Command Prompt (you can search for cmd in the Start menu).
    • Run the following command to install Jupyter: 
      pip install jupyter

  2. Run Jupyter Notebook:

    • After installation, you can start Jupyter Notebook by running the following command in Command Prompt: 
      jupyter notebook
    • A new browser window will open, displaying the Jupyter interface. You can now create new notebooks and start coding.
